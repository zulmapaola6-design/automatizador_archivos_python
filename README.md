#  Automatizador de Archivos en Python

Este proyecto es un script en Python que organiza automáticamente archivos dentro de una carpeta, clasificándolos por tipo y moviéndolos a carpetas específicas.

Es ideal para automatizar tareas repetitivas como la organización de descargas o documentos personales.

---

##  Funcionalidades

- Organiza archivos automáticamente según su extensión
- Crea carpetas si no existen
- Clasifica archivos PDF, imágenes, documentos y otros
- Código simple, claro y fácil de modificar
